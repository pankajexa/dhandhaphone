# DhandhaPhone Knowledge Base

Reference guide for all domain knowledge available to the AI agent.
Use this index to find the right knowledge file for a user's question.

## Directory Map

### `/knowledge/gst/`
GST (Goods and Services Tax) compliance, rates, filing, and credits.
**Reference when:** User asks about tax rates, GSTIN, filing deadlines, ITC, composition scheme.

| File | Topic |
|------|-------|
| `_overview.md` | GST basics, registration thresholds, GSTIN format |
| `rates-goods.md` | Tax rate brackets (0-28%) with goods examples and HSN codes |
| `gstr-filing.md` | Return types, due dates, late fees, filing process |
| `input-credit.md` | Input Tax Credit eligibility, blocked credits, reconciliation |
| `composition-scheme.md` | Simplified scheme for small businesses under 1.5Cr |

### `/knowledge/indian-business/`
Indian SMB context: culture, seasons, festivals, credit norms.
**Reference when:** User needs business advice, seasonal planning, credit management, regional context.

| File | Topic |
|------|-------|
| `_overview.md` | Indian MSME landscape, key characteristics |
| `festival-calendar.md` | Pan-Indian + regional festivals with business impact |
| `credit-culture.md` | Udhaar/credit norms, payment cycles, collection strategies |
| `seasonal-patterns.md` | Four business seasons, impact on inventory and cash flow |
| `regional-customs.md` | Business norms by region (North/South/West/East India) |

### `/knowledge/inventory/`
Stock management for small shops and businesses.
**Reference when:** User asks about reordering, stock levels, expiry, wastage.

| File | Topic |
|------|-------|
| `_overview.md` | Why track inventory, basic metrics, approaches by business type |
| `reorder-logic.md` | Reorder point formula, safety stock, lead time calculations |
| `shelf-life.md` | Perishable categories, FIFO, markdown strategies |
| `fifo-basics.md` | First In First Out method, physical implementation |

### `/knowledge/pricing/`
Pricing, margins, and competitive positioning.

| File | Topic |
|------|-------|
| `_overview.md` | Pricing strategies: cost-plus, competitive, value-based |
